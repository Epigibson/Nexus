# Security Policy

## Supported Versions

Nexus CLI and Dashboard are actively maintained. Currently, we provide security updates for the following versions:

| Version | Supported          |
| ------- | ------------------ |
| v1.x.x  | :white_check_mark: |
| < v1.0  | :x:                |

## Reporting a Vulnerability

We take the security of Nexus very seriously. If you discover a vulnerability, **please do not open a public issue.**

Instead, please use the **Private Vulnerability Reporting** feature in GitHub:
1. Go to the **Security** tab of this repository.
2. Click on **Advisories** on the left sidebar.
3. Click on the **Report a vulnerability** button.

We will review your report and respond as quickly as possible. We appreciate your efforts to responsibly disclose your findings, and we will make every effort to acknowledge your contributions.
