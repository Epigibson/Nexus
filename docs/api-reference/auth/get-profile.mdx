---
title: "Get Profile"
api: "GET /api/v1/auth/me"
description: "Obtener el perfil del usuario autenticado."
auth: "bearer"
---

<RequestExample>
```bash
curl http://localhost:8000/api/v1/auth/me \
  -H "Authorization: Bearer {token}"
```
</RequestExample>

<ResponseExample>
```json 200
{
  "id": "user-1",
  "email": "dev@acme-corp.com",
  "display_name": "Carlos Dev",
  "avatar_url": null,
  "plan": "free",
  "created_at": "2026-04-04T07:15:29"
}
```
</ResponseExample>
