## Compilar desde código fuente

```bash
cd core/
go build -o nexus ./cmd/main.go
```

## Agregar al PATH

<Tabs>
  <Tab title="macOS / Linux">
    ```bash
    sudo mv nexus /usr/local/bin/
    nexus version
    ```
  </Tab>
  <Tab title="Windows">
    ```powershell
    Copy-Item nexus.exe C:\bin\
    # Agregar C:\bin a la variable PATH del sistema
    nexus version
    ```
  </Tab>
</Tabs>

## Verificar

```bash
nexus version
# Nexus v0.1.0
```
